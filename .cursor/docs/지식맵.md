## 🎯 Part 2: 과제 완료 후 얻게 될 지식

### 1. React Hook 심화

#### 학습 성과
- [x] **Custom Hook 설계 능력**
  - 비즈니스 로직을 Hook으로 추출
  - Hook의 단일 책임 원칙 적용
  - Hook 간 의존성 관리

- [x] **Hook 최적화 전략**
  - `useCallback`과 `useMemo` 적절한 사용
  - 불필요한 리렌더링 방지
  - 성능 최적화 패턴

- [x] **Hook 테스트 가능한 구조 설계**
  - Hook을 독립적으로 테스트 가능하게 설계
  - 의존성 주입 패턴 이해

#### 실전 적용 예시
```typescript
// 과제 전: 컴포넌트에 모든 로직 포함
const Cart = () => {
  const [cart, setCart] = useState([]);
  // 복잡한 로직들이 컴포넌트에 직접...
};

// 과제 후: Hook으로 분리된 깔끔한 구조
const Cart = () => {
  const { cart, addItem, removeItem } = useCart();
  const total = useCartTotal(cart);
  // 컴포넌트는 UI 렌더링에만 집중
};
```

### 2. 함수형 프로그래밍 실전

#### 학습 성과
- [x] **순수함수 작성 능력**
  - 부수 효과 없는 함수 설계
  - 테스트하기 쉬운 함수 작성
  - 함수 조합(composition) 활용

- [x] **불변성 관리 능력**
  - 상태 업데이트 시 불변성 유지
  - 깊은 객체/배열 업데이트 패턴
  - 성능과 안정성 균형

- [x] **액션과 계산 분리**
  - 상태 변경 로직과 계산 로직 구분
  - 순수함수와 액션의 명확한 분리

#### 실전 적용 예시
```typescript
// 과제 전: 부수 효과가 있는 함수
let total = 0;
const calculate = (items) => {
  total = 0; // 외부 변수 수정
  items.forEach(item => total += item.price);
  return total;
};

// 과제 후: 순수함수
const calculateTotal = (items: CartItem[]): number => {
  return items.reduce((sum, item) => sum + item.price, 0);
};
```

### 3. 아키텍처 설계 능력

#### 학습 성과
- [x] **엔티티 기반 분리 능력**
  - 엔티티를 다루는 코드와 UI 코드 구분
  - 도메인 로직과 프레젠테이션 로직 분리
  - 재사용 가능한 컴포넌트 설계

- [x] **계층 구조 설계 능력**
  - entities → features → UI 계층 구조 이해
  - 의존성 방향 관리 (상위 → 하위)
  - 순환 의존성 방지

- [x] **모듈화 능력**
  - 단일 책임 원칙 적용
  - 높은 응집도, 낮은 결합도 달성
  - 코드 재사용성 향상

#### 실전 적용 예시
```
과제 전 구조:
src/
  App.tsx (모든 로직 포함)

과제 후 구조:
src/
  entities/
    cart.ts
    product.ts
  features/
    useCart.ts
    useProduct.ts
    calculateTotal.ts
  components/
    CartView.tsx
    ProductCard.tsx
  App.tsx
```

### 4. 상태 관리 심화

#### 학습 성과 (심화과제)
- [x] **Context API 활용 능력**
  - 전역 상태 관리 설계
  - Provider와 Consumer 패턴
  - Context 최적화 (불필요한 리렌더링 방지)

- [x] **Jotai 활용 능력** (선택)
  - Atom 기반 상태 관리
  - 파생 상태(derived state) 관리
  - 비동기 상태 처리

- [x] **Props Drilling 해결 능력**
  - 어떤 Props를 전역 상태로 올릴지 판단
  - 도메인 Props vs 전역 Props 구분
  - Props 전달 최적화

#### 실전 적용 예시
```typescript
// 과제 전: Props Drilling
<App user={user}>
  <Header user={user} />
  <Main user={user}>
    <ProductList user={user} />
  </Main>
</App>

// 과제 후: Context 활용
<UserProvider>
  <App>
    <Header /> {/* useUser() Hook 사용 */}
    <Main>
      <ProductList /> {/* useUser() Hook 사용 */}
    </Main>
  </App>
</UserProvider>
```

## 🎓 학습 경로 요약

### 단계별 학습 흐름

```
1단계: React 기초
  ↓
2단계: Hook 이해 및 Custom Hook 작성
  ↓
3단계: 함수형 프로그래밍 패턴 적용
  ↓
4단계: 엔티티 기반 분리 및 계층 구조 설계
  ↓
5단계: 전역 상태 관리 (Context/Jotai)
  ↓
6단계: 전체 아키텍처 통합 및 최적화
```

### 핵심 학습 포인트

1. **분리 (Separation)**
   - 로직과 UI 분리
   - 엔티티와 UI 분리
   - 액션과 계산 분리

2. **추상화 (Abstraction)**
   - Hook으로 로직 추상화
   - 컴포넌트로 UI 추상화
   - 함수로 계산 추상화

3. **구조화 (Structure)**
   - 계층 구조 설계
   - 모듈 구조 설계
   - 의존성 구조 설계
